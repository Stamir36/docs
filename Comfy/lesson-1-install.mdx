## 1. Что такое ComfyUI?

ComfyUI — это **программа для генерации изображений с помощью нейросетей**, в частности **Stable Diffusion**.  
Она использует **нодовый интерфейс**: вы соединяете блоки (ноды), и каждая часть схемы выполняет свою задачу.

![Пример нодов](/images/ComfyUI/IMG_001.png)

## 2. Установка ComfyUI
<Info>
  Это локальный запуск. Запуск на сервере - будет в следующем разделе.
</Info>

<Steps>
  <Step title="Скачайте ComfyUI">
    Перейдите на [сайт документации ComfyUI](https://docs.comfy.org/installation/comfyui_portable_windows) и скачайте последнюю portable версию.

    ![Скачивание Portable-версии](/images/ComfyUI/IMG_002.png)
  </Step>

  <Step title="Распакуйте архив">
    Распакуйте скачанный архив в удобное место на вашем компьютере, например в `C:/ComfyUI`.
  </Step>

  <Step title="Установите зависимости">
    В папке ComfyUI найдите файл `run_nvidia_gpu.bat` (для Windows с видеокартой NVIDIA) или `run_cpu.bat` (для запуска на процессоре).

    При первом запуске программа сама установит необходимые библиотеки (Python и PyTorch).

    <Tip>
      Если у вас **AMD видеокарта** или **Linux/MacOS**, следуйте инструкции из README на GitHub.
    </Tip>
  </Step>
</Steps>

---

## 3. Первый запуск

<Steps>
  <Step title="Запустите файл">
    Дважды кликните по `run_nvidia_gpu.bat` (или `run_cpu.bat`). Откроется консольное окно.
  </Step>

  <Step title="Откройте интерфейс">
    После загрузки откройте в браузере адрес:
    ```
    http://127.0.0.1:8188
    ```
    Вы увидите интерфейс ComfyUI.

    ![Скриншот интерфейса ComfyUI после первого запуска](/images/ComfyUI/IMG_003.png)
  </Step>
</Steps>

---

## 4. Возможные проблемы

<AccordionGroup>
  <Accordion title="Окно сразу закрывается">
    Убедитесь, что у вас установлены **драйверы для видеокарты** и **Python 3.10+**.
  </Accordion>

  <Accordion title="Ошибка при установке библиотек">
    Проверьте подключение к интернету и попробуйте запустить `update.bat` внутри папки ComfyUI.
  </Accordion>

  <Accordion title="Проблемы с GPU">
    Запустите версию для CPU (`run_cpu.bat`) — она медленнее, но подходит для любых компьютеров.
  </Accordion>
</AccordionGroup>
